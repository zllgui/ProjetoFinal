A força é uma das sete estatísticas primárias do sistema SPECIAL.

A força é principalmente relevante para duas mecânicas no jogo: Carregar Peso e satisfazer os requisitos mínimos de Força em armas. Cada ponto dele concede 25 lbs. de Carry Weight Não cumprir o requisito mínimo de Força de uma arma penaliza a precisão do alvo com essa arma em -20% para cada ponto perdido em Força.

"Força física bruta. Uma força alta é boa para personagens físicos." - Guia de sobrevivência dos Ermos.
